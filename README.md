# Проект Mesto
Сервис Mesto в Яндекс.Практикуме: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.


## Технологии
* JavaScript
* CSS
* HTML
* Адаптивная верстка
* БЭМ
